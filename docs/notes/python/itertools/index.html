    # Python itertools 模块方法介绍#

    itertools 模块提供了一系列用于高效循环的迭代器函数，以下是主要方法的简单介绍：

    无限迭代器#

    count(start=0, step=1)#

    NOTE

    从 start 开始，以 step 为步长无限计数

    count(10) → 10, 11, 12, 13, ...

    cycle(iterable)#

    NOTE

    无限重复迭代器中的元素

    cycle('ABCD') → A B C D A B C D ...

    repeat(object[, times])#

    NOTE

    重复对象指定次数或无限重复

    repeat(10, 3) → 10, 10, 10

    有限迭代器#

    accumulate(iterable[, func])#

    NOTE

    返回累积值（默认求和）

    accumulate([1,2,3,4,5]) → 1, 3, 6, 10, 15

    chain(*iterables)#

    NOTE

    将多个迭代器连接成一个

    chain('ABC', 'DEF') → A B C D E F

    chain.from_iterable(iterable)#

    NOTE

    从可迭代对象中链式连接

    chain.from_iterable(['ABC', 'DEF']) → A B C D E F

    compress(data, selectors)#

    NOTE

    根据选择器筛选数据

    compress('ABCDEF', [1,0,1,0,1,1]) → A C E F

    dropwhile(predicate, iterable)#

    NOTE

    当 predicate 为真时丢弃元素，然后返回剩余所有

    dropwhile(lambda x: x<5, [1,4,6,4,1]) → 6, 4, 1

    filterfalse(predicate, iterable)#

    NOTE

    返回 predicate 为假的元素

    filterfalse(lambda x: x%2, range(10)) → 0, 2, 4, 6, 8

    groupby(iterable, key=None)#

    NOTE

    按键分组相邻的重复元素

    [k for k, g in groupby('AAAABBBCCDAABBB')] → A B C D A B

    islice(iterable, stop) / islice(iterable, start, stop[, step])#

    NOTE

    对迭代器进行切片

    islice('ABCDEFG', 2, None) → C D E F G

    starmap(function, iterable)#

    NOTE

    将参数从迭代器中解包后应用函数

    starmap(pow, [(2,5), (3,2), (10,3)]) → 32, 9, 1000

    takewhile(predicate, iterable)#

    NOTE

    当 predicate 为真时获取元素

    takewhile(lambda x: x<5, [1,4,6,4,1]) → 1, 4

    tee(iterable, n=2)#

    NOTE

    将一个迭代器拆分为 n 个独立的迭代器

    a, b = tee(range(5))

    zip_longest(*iterables, fillvalue=None)#

    NOTE

    类似于 zip，但以最长迭代器为准，用 fillvalue 填充

    zip_longest('ABCD', 'xy', fillvalue='-') → Ax By C- D-

    组合迭代器#

    product(*iterables, repeat=1)#

    NOTE

    计算笛卡尔积

    product('AB', repeat=2) → AA AB BA BB

    permutations(iterable, r=None)#

    NOTE

    返回长度为 r 的所有排列

    permutations('ABCD', 2) → AB AC AD BA BC BD CA CB CD DA DB DC

    combinations(iterable, r)#

    NOTE

    返回长度为 r 的所有组合（按输入顺序）

    combinations('ABCD', 2) → AB AC AD BC BD CD

    combinations_with_replacement(iterable, r)#

    NOTE

    返回包含重复元素的组合

    combinations_with_replacement('ABC', 2) → AA AB AC BB BC CC
